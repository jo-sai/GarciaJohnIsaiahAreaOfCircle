# Circle Area Calculator

A straightforward Python script that calculates the area of a circle based on user input, demonstrating the use of Python's built-in `math` module.

## 📝 Description

This repository contains a simple Python program that computes the area of a circle. It prompts the user to input the radius, utilizes `math.pi` for accurate calculations, and outputs the final area in two formats: rounded to two decimal places and the exact float value. It is a great basic example of handling floating-point user input and arithmetic operations.

## Features

* **Math Module Integration:** Imports Python's built-in `math` library to use the exact value of Pi (π).
* **User Input:** Dynamically accepts the circle's radius as a floating-point number.
* **Precision Formatting:** Displays the calculated area cleanly rounded to two decimal places, alongside the full unrounded result.
